## Troubleshooting

### Response Not Appearing in GUI
- Check: `tail -1 local-agent-vscode/ipc/outbox.jsonl`
- Should show recent response

### File Not Created
- Check: `ls -la /tmp/` | grep test
- Check agent log: `tail -50 agent.log`
- Look for error messages

### Code Quality Issues
- Agent's code generation depends on model
- openchat is good for basic-to-intermediate code
- More complex tasks may need refinement

### Too Slow
- First request: 10-15 seconds (model loading)
- Subsequent: 5-8 seconds
- Complex tasks: 15-30 seconds
